Un petit logo **IPX7** sur la boîte d'un sextoy promet 30 minutes d'immersion sous 1 mètre d'eau sans dommage. Concrètement, ça change quoi pour l'usage sous la douche, dans le bain, ou simplement au moment du nettoyage ? Décryptage de cet indice trop souvent ignoré à l'achat, et comparaison avec les autres certifications du marché.

![Indice IPX7 sextoy](/images/products/lelo-sona-2.jpg "IPX7 sextoy")

## À retenir

- **IPX7** = immersion 30 min à 1 mètre sans dégât
- **Standard** des sextoys premium modernes (Womanizer, Lelo, We-Vibe)
- **Permet** : douche, bain, nettoyage à l'eau
- **Ne permet pas** : plongée, baignade longue (chercher IPX8)
- **À vérifier** : la mention exacte sur l'emballage ou la fiche technique
- **Modèles : ** Lelo Sona 2, Womanizer Premium 2, Satisfyer Pro 2, Lovense Lush 3

## Que signifie IPX7

L'indice IP (Ingress Protection) est défini par la norme internationale IEC 60529. Il se décompose en deux chiffres :

- **Premier chiffre** : protection contre les solides (poussières)
- **Second chiffre** : protection contre les liquides

Pour les sextoys, le premier chiffre est généralement remplacé par "X" (pas testé pour les solides). Le second indique la résistance à l'eau.

### L'échelle complète

| Indice | Protection liquide | Usage |
|---|---|---|
| IPX0 | Aucune | Pas d'eau |
| IPX1 | Gouttes verticales | Pas d'usage humide |
| IPX2 | Gouttes en biais | Pas d'usage humide |
| IPX3 | Pluie | Pas d'usage humide |
| IPX4 | Éclaboussures | Lavage rapide possible |
| IPX5 | Jets d'eau | Nettoyage sous l'eau |
| IPX6 | Jets puissants | OK sous la douche |
| IPX7 | Immersion 30 min à 1 m | OK sous l'eau, dans le bain |
| IPX8 | Immersion prolongée | Piscine, plongée |

### Pour un sextoy

- **IPX4 minimum** : pour qu'un nettoyage à l'eau soit possible sans risque
- **IPX5/6** : possible usage sous la douche
- **IPX7** : standard recommandé, permet immersion partielle
- **IPX8** : luxe, permet immersion prolongée

## Pourquoi IPX7 est le standard

### Hygiène facilité

Avec IPX7, on peut rincer le toy directement sous l'eau ou au robinet, sans craindre l'infiltration. Nettoyage rapide et complet. Voir notre [guide nettoyage](/blog/comment-nettoyer-son-sextoy/).

### Usage sous la douche

L'IPX7 permet l'usage sous la douche sans contrainte. Le toy résiste aux éclaboussures, jets, immersion partielle. Voir notre [guide sextoy sous la douche](/blog/sextoy-sous-la-douche/).

### Bain confortable

Pour un usage dans le bain, IPX7 est le minimum. Si vous immergez le toy plus de 30 minutes, viser IPX8.

### Sécurité

Une légère étanchéité défaillante peut court-circuiter la batterie lithium-ion. IPX7 garantit la fiabilité dans les conditions d'usage normales.

## Comment vérifier l'IPX7

### Sur l'emballage

Mention claire "IPX7" ou "100 % waterproof" sur la boîte. Si "splash-proof" sans précision = IPX4 max, à éviter pour usage immergé.

### Sur la fiche technique en ligne

Site du fabricant détaille l'indice IP exact. Vérifier avant achat.

### Sur l'app (modèles connectés)

L'app Lovense Remote et We-Connect indiquent souvent l'indice IP dans les caractéristiques produit.

### Symbole sur le toy

Petit pictogramme d'eau parfois gravé sur le toy. Pas systématique mais bonne indication.

## Tests pour vérifier l'IPX7 réel

### Test prudent

1. Plonger le toy dans 5 cm d'eau pendant 2 minutes
2. Sortir, sécher l'extérieur
3. Tester le fonctionnement immédiatement
4. Si OK, passer au test complet

### Test complet (sur toy déjà en service)

1. Plonger le toy dans un verre d'eau 30 minutes
2. Vérifier qu'aucune buée n'apparaît à l'intérieur
3. Sécher, attendre 24 heures
4. Tester le fonctionnement

Si le toy fonctionne après ces tests = IPX7 fiable. Sinon, étanchéité défaillante, contacter le SAV.

## Et les modèles sans IPX

Quelques sextoys n'ont aucune certification IPX. Ils sont souvent :

- Modèles anciens (avant 2015)
- Modèles très bas de gamme (PVC sans certifs)
- Modèles non électroniques (godemiches simples, plugs en silicone seul) qui n'ont pas besoin d'IPX par construction

### Pour les modèles non IPX

Nettoyage par essuyage humide uniquement. Ne pas immerger.

## Différence IPX7 vs IPX8

| Critère | IPX7 | IPX8 |
|---|---|---|
| Durée immersion | 30 min | Illimitée (selon constructeur) |
| Profondeur | 1 m | > 1 m (selon constructeur) |
| Usage douche | OK | OK |
| Usage bain | OK | OK |
| Usage piscine | À éviter (chlore) | OK |
| Plongée | Non | Selon constructeur |

Pour 99 % des usages, IPX7 suffit largement. IPX8 est un peu marketing pour la plupart des utilisateurs.

## Questions fréquentes

<details>
<summary>Mon sextoy IPX7 a pris l'eau, c'est normal</summary>

Non. Un IPX7 doit résister à 30 min d'immersion à 1 m. Si de l'eau pénètre, l'étanchéité est défaillante. Sous garantie : SAV. Hors garantie : remplacement obligatoire (court-circuit possible).
</details>

<details>
<summary>Peut-on faire confiance à la mention IPX7</summary>

Sur les marques sérieuses (Womanizer, Lelo, We-Vibe, Satisfyer), oui, c'est testé en usine. Sur les marques inconnues bas de gamme, méfiance, la mention peut être marketing sans test réel.
</details>

<details>
<summary>L'IPX7 dégrade-t-il avec le temps</summary>

Oui, légèrement. Les joints d'étanchéité s'usent. Un toy IPX7 utilisé 5 ans peut ne plus être pleinement étanche. Vérifier régulièrement.
</details>

<details>
<summary>Quelle différence entre IPX7 et "100% waterproof"</summary>

"100 % waterproof" est marketing, pas une norme. Le terme exact est IPX7 ou IPX8. Si vous voyez "100 % waterproof" sans indice IP, demander le détail au fabricant.
</details>
